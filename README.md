# Комплексный анализ продаж интернет-магазина

## 1. Описание проекта
**Название проекта:** Комплексный анализ продаж интернет-магазина

**Цель проекта:**  
Выявить точки роста выручки и улучшить понимание клиентской базы для формирования эффективной маркетинговой и товарной стратегии.

**Задачи проекта:**
- Определить месяцы с наибольшим количеством проданных товаров.
- Определить месяцы с наибольшей суммой продаж.
- Выявить страны-лидеры по количеству проданных товаров.
- Выявить страны-лидеры по сумме продаж.
- Определить товары-лидеры по продажам и товары, не пользующиеся спросом.
- Рассчитать средний доход от клиента (ARPU).
- Провести RFM-анализ для сегментации клиентской базы.
- Построить прогноз продаж на основе исторических данных.

## 2. Источники данных
В проекте использован набор данных о транзакциях в сфере розничной онлайн-торговли за 2010–2011 гг.

**Характеристики датасета:**
- Количество записей: более 5 000.
- Количество полей: 10.

**Описание полей:**
| Название поля | Описание |
| :--- | :--- |
| `InvoiceNo` | Номер заказа |
| `StockCode` | Код продукта |
| `Description` | Описание продукта |
| `Quantity` | Количество товара в заказе |
| `InvoiceDate` | Дата и время заказа |
| `UnitPrice` | Цена за единицу (фунты стерлингов) |
| `CustomerID` | Идентификатор клиента |
| `Country` | Название страны, где находится покупатель |

## 3. Методология и инструменты
В ходе выполнения проекта использовались следующие инструменты и библиотеки:

- **Python** — основной язык анализа.
  - `pandas`, `numpy` — обработка и трансформация данных.
  - `matplotlib`, `seaborn` — визуализация данных.
  - `statsmodels` — прогнозирование временных рядов.
- **SQL** — выполнение предварительных запросов к данным.
- **Power BI** — создание интерактивного дашборда для визуализации ключевых метрик.

## 4. Что сделано в ходе работы
- Проведен **сезонный и географический анализ** продаж для определения пиковых периодов и регионов с наибольшим потенциалом.
- Выполнен **корреляционный анализ** факторов, влияющих на выручку.
- Рассчитан **средний доход от клиента (ARPU)** и проведена **RFM-сегментация** для выделения наиболее ценных групп пользователей и групп, требующих удержания.
- Построен **прогноз продаж** на основе исторических данных с учетом сезонности.
- Разработан **интерактивный дашборд в Power BI** для визуализации ключевых инсайтов.

## 5. Результаты и выводы

### 5.1. Анализ продаж по месяцам
- **Количество проданных товаров:**  
  Наибольшее количество продаж приходится на **ноябрь**, затем следуют **сентябрь и октябрь**.  
  Наблюдается резкое падение в декабре. Основной объем продаж сосредоточен во второй половине года.

- **Сумма продаж:**  
  Самые высокие суммы выручки зафиксированы в **ноябре**, далее — в **сентябре и октябре**.  
  Динамика суммы продаж совпадает с динамикой количества проданных товаров.

- **Рекомендация:**  
  Для повышения продаж в месяцы спада (декабрь и первая половина года) рекомендуется обновить ассортимент товарами, актуальными в этот период, и скорректировать рекламную стратегию.

### 5.2. Географический анализ
- **Страны-лидеры по количеству и сумме продаж:**  
  **Великобритания** значительно опережает другие страны по обоим показателям.  
  Покупатели из Великобритании являются основной целевой аудиторией.

### 5.3. Анализ товаров
- Выявлено **15 товаров**, которые не приносили доход на протяжении двух лет.
- Обнаружено **27 товаров с отрицательной суммой продаж** (возвраты).
- **Рекомендация:**  
  Рассмотреть возможность отказа от продажи нерентабельных позиций. Если причина низких продаж — недостаточное понимание характеристик продукта, стоит улучшить его описание. Торговлю ходовыми позициями следует сделать более активной.

### 5.4. Прогнозирование продаж
- Прогноз построен на **6 месяцев вперед** с учетом праздников и недельной сезонности.
- График прогноза демонстрирует **потенциальный рост количества заказов**, что позволяет оптимизировать товарные запасы и бюджет.

### 5.5. RFM-анализ и средний доход от клиента
- **ARPU (Average Revenue Per User):** Рассчитан средний доход на одного клиента, что позволило оценить общую ценность клиентской базы.
- **RFM-сегментация:** Клиенты разделены на группы (например, «лояльные», «спящие», «новые», «риск потери»). Это позволило предложить точечные стратегии удержания и увеличения LTV (Lifetime Value).

## 6. Общий вывод и рекомендации
В результате проведенного анализа были сформулированы следующие ключевые рекомендации для бизнеса:

1.  **Фокус на сезонность:** Сконцентрировать маркетинговые бюджеты на сентябрь–ноябрь, а для стимулирования спада в декабре предложить специальные акции или сезонный ассортимент.
2.  **Географический фокус:** Усилить работу с рынком Великобритании, а также протестировать рекламные кампании в странах, показавших следующий по величине потенциал.
3.  **Ассортиментная политика:** Исключить из продаж товары с устойчиво низким спросом и отрицательной доходностью. Улучшить карточки товаров для позиций, где возможна проблема с восприятием характеристик.
4.  **Работа с клиентами:** Внедрить стратегии удержания для сегментов «риск потери» и программы лояльности для сегмента «золотые клиенты», выявленных в ходе RFM-анализа.
5.  **Планирование:** Использовать полученную прогнозную модель для планирования закупок и управления складскими запасами.
